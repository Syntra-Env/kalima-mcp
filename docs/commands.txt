Kalima CLI quick reference

Navigate / read:
  read chapters
  read chapter <N>
  read <surah:ayah>
  read verse <N>        (uses current surah context)
  read word <N>
  read sentence <N>
  read morpheme <letter>
  read letter <N>

Layers (token overlays):
  layer                (show current + list)
  layer <number|name>  (e.g. layer root, layer pos, layer dependency)
  layer next | prev

Other:
  history              (click an entry to re-run)
  clear
  status
  legend
  help
  exit | quit

Concordance (Query Mode):
  Ctrl+Q               (enter Query Mode)
  q | query | concordance  (enter Query Mode as a command)
  Alt+O/R/L/P/C/G      (switch layer: Original/Root/Lemma/POS/Case/Gender)
  click token          (add/update constraint for next anchor)
  Shift+click token    (toggle constraint on/off)
  Shift+hover token    (show current layer label)
  Ctrl+Z               (remove last anchor)
  ?                    (show Query Mode help hint when query is empty)
  Enter                (run concordance search)
  Esc                  (exit Query Mode without running)

Query syntax:
  #N key:value ...     (anchors; one token per anchor, matched sequentially)
  Values with spaces:  r:\"...\"  (quoted values supported, e.g. r:\"ع ج ل\")
  OR values:           g:M|F   (also supports commas: g:M,F)
  Wildcards:           r:عجل*  (* expands as wildcard)
  Negation:            !g:M
  Gap (distance):      ~0 (adjacent), ~1-3 (allow 1..3 tokens between anchors)
