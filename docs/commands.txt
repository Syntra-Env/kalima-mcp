Kalima CLI quick reference

Navigate / read:
  read chapters
  read chapter <N>
  read <surah:ayah>
  read verse <N>        (uses current surah context)
  read word <N>
  read sentence <N>
  read morpheme <letter>
  read letter <N>

Layers (token overlays):
  layer                (show current + list)
  layer <number|name>  (e.g. layer root, layer pos, layer dependency)
  layer next | prev

Other:
  history              (click an entry to re-run)
  clear
  status
  legend
  help
  exit | quit
